About CloudBoot Packages (.cbp)

CloudBoot Package files are encrypted, vendor-specific hardware configuration tools. They enable CloudBoot to manage RAID controllers, BIOS settings, and other hardware components from different manufacturers.

Security: All providers run in isolated sandboxes and are automatically cleaned from memory after execution.
